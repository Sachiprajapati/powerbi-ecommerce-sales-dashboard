# E-Commerce Sales Dashboard (Power BI)

## Overview

This project presents an interactive Power BI dashboard built using e-commerce sales data. The dashboard provides insights into sales performance, profit trends, product categories, sub-categories, and payment methods.

## Features

* Sales, Profit, Quantity, and Orders KPI Cards
* Monthly Profit Analysis
* Category-wise Sales Distribution
* Sub-category Profit Analysis
* Payment Mode Analysis
* Interactive Filters and Slicers

## Tools Used

* Power BI Desktop
* Data Visualization

## Dataset

The dataset contains e-commerce sales transactions including sales amount, quantity, profit, category, sub-category, and payment mode information.


## Key Insights

* Clothing category contributed the highest quantity of sales.
* COD was the most frequently used payment mode.
* Profit showed strong performance in January and November.
* Some months recorded negative profit indicating potential business improvement areas.
